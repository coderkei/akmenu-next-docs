    Setting Up AKMenu-Next

    Info

    AKMenu-Next is a frontend for nds-bootstrap & Pico-Loader based on a fork of akmenu4 for running Nintendo DS Software & Homebrew. It supports cheats along with support for Acekard themes. For controls such as soft-reset and quitting games, please see nds-bootstrap controls

    Setup Guide:

    Installing AKMenu-Next Flashcart Edition

    1. Download the latest release of AKMenu-Next Flashcart Edition.

    2. Extract the downloaded akmenu-next-flashcart.zip file with 7-Zip.

    3. From within the akmenu-next files, copy the following files/folders to your SD card root:

      • _nds folder
      • _pico folder
      • BOOT.NDS

    4. Download the latest release of nds-bootstrap.

    5. Extract the downloaded nds-bootstrap.zip file with 7-Zip.

    6. Copy the contents of the files from the extracted nds-bootstrap.zip file to the _nds folder on your SD card.

    Pico-Loader

    If you would like to also use Pico-Loader please follow the below steps, otherwise skip this section. Please bear in mind that Pico-Loader currently lacks soft-reset. Any bugs should be reported on the Pico-Loader Issues Page.

    Warning

    Using the wrong version of Pico-Loader could damage your flashcard! If you are unsure which version to use, check the Supported Platforms section.

    1. Download the latest release of Pico-Loader. Choose the version that corresponds with your flashcart.

    2. Extract the downloaded Pico_Loader__YOUR_FLASHCART_HERE.zip file with 7-Zip.

    3. Copy the contents of the files from the extracted Pico_Loader_YOUR_FLASHCART_HERE.zip file to the _pico folder on your SD card. Create this folder if it does not exist.

    4. This loader can be selected by going to the settings in AKMenu-Next and finding the option labeled Game Loader and changing the setting to Pico Loader.

    You can now load BOOT.NDS with your flashcart's kernel, or alternatively follow the autoboot steps below.

    Autobooting AKMenu-Next

    Note

    If you don't see your flashcart in the Autoboot folder or the corresponding one in the README.TXT file, skip this section as you will need to use your flashcart kernel to load AKMenu-Next instead. In some cases you can make a copy of BOOT.NDS and rename it to default.nds to make it chainload autoboot on some flashcart kernels.

    1. Open the Autoboot folder within the AKMenu-Next files. Find the folder that corresponds to your flashcart name or type.

    2. Copy the contents of the folder (do not copy the folder itself) that corresponds with your flashcart to the root of your SD card.

    3. If your flashcart is a DSTTi clone that requires additional boot files, check the Supplementary DSTTi Clone Files folder. You can check the YSMenu compatibility list to check which one you need. If the file you need is not there, you can make copies of the TTMenu.dat file from the DSTT & DSTTi folder and rename them accordingly.

    Cheats

    1. If you'd like to be able to use cheats on your games, download a cheat database.

    2. You will need the usrcheat.dat file from the download link in the post. Copy this file to _nds/akmenunext/cheats/ on your SD card. (Create the cheats folder if it doesn't exist)

    Installing AKMenu-Next DSpico Edition

    1. Download the latest release of AKMenu-Next DSpico Edition.

    2. Extract the downloaded akmenu-next-pico.zip file with 7-Zip.

    3. From within the akmenu-next files, copy the following files/folders to your SD card root:

      • _nds folder
      • _pico folder
      • BOOT.NDS
      • _picoboot.nds (If you wish to autoboot AKMenu-Next)

    4. Download the latest release of Pico-Loader for DSpico.

    5. Extract the downloaded Pico_Loader_DSPICO.zip file with 7-Zip.

    6. Copy the contents of the files from the extracted Pico_Loader_DSPICO.zip file to the _pico folder on your SD card.

    7. This loader can be selected by going to the settings in AKMenu-Next and finding the option labeled Game Loader and changing the setting to Pico Loader.

    nds-bootstrap (Optional)

    Install nds-bootstrap additionally if you wish to use soft reset or a game not compatible with Pico-Loader. Please note that you won't benefit from some of the features offered by the DSpico if you choose to use nds-bootstrap.

    1. Download the latest release of nds-bootstrap.

    2. Extract the downloaded nds-bootstrap.zip file with 7-Zip.

    3. Copy the contents of the files from the extracted nds-bootstrap.zip file to the _nds folder on your SD card.

    4. This loader can be selected by going to the settings in AKMenu-Next and finding the option labeled Game Loader and changing the setting to nds-bootstrap.

    You can now load BOOT.NDS from Pico-Launcher kernel, or alternatively autoboot by using the _picoboot.nds file.

    Cheats

    1. If you'd like to be able to use cheats on your games, download a cheat database.

    2. You will need the usrcheat.dat file from the download link in the post. Copy this file to _nds/akmenunext/cheats/ on your SD card. (Create the cheats folder if it doesn't exist)

    Running DSiWare (DSi & 3DS only)

    Please see this guide for running DSiWare on a DSpico.

    Note

    This requires a modded Nintendo DSi system. If your Nintendo DSi is not modded, please follow dsi.cfw.guide to install it.

    Installing AKMenu-Next Nintendo DSi Edition

    1. Download the latest release of AKMenu-Next DSi Edition.

    2. Extract the downloaded akmenu-next-dsi.zip file with 7-Zip.

      Warning

      If you are using TWiLight Menu++ and want to dual boot AKMenu-Next, do not copy over BOOT.NDS as this will overwrite the one TWiLight Menu++ uses. You can instead launch akmenu-next.dsi with TWiLight Menu++ or Unlaunch.

    3. From within the AKMenu-Next files, copy the following files/folders to your SD card root:

      • _nds folder
      • BOOT.NDS (Please read the warning above)
      • akmenu-next.dsi
      • title folder (if using HiyaCFW)

    4. Download the latest release of nds-bootstrap.

    5. Extract the downloaded nds-bootstrap.zip file with 7-Zip.

    6. Copy the contents of the files from the extracted nds-bootstrap.zip file to the _nds folder on your SD card.

    7. Load BOOT.NDS with your desired entrypoint method (e.g. Memory Pit, stylehax or Flipnote Lenny). Alternatively, use Unlaunch or TWiLight Menu++ to boot akmenu-next.dsi. If you are using HiyaCFW, it will appear as an app on your home screen, provding that you copied the title folder.

    Cheats

    1. If you'd like to be able to use cheats on your games, download a cheat database.

    2. You will need the usrcheat.dat file from the download link in the post. Copy this file to _nds/akmenunext/cheats/ on your SD card. (Create the cheats folder if it doesn't exist)

    Autobooting AKMenu-Next

    1. Turn on your DSi while holding A and B

    2. In the Unlaunch menu, go to OPTIONS

    3. Set NO BUTTON or a button of your choice to the akmenu-next entry that says BOOT.NDS (or akmenu-next.dsi if applicable) on the bottom screen

    Note

    This requires a modded Nintendo 3DS system. If your Nintendo 3DS is not modded, please follow 3ds.hacks.guide to install it. This guide is also applicable to the Nintendo 2DS models but will be referenced as a 3DS in the steps below.

    Installing AKMenu-Next Nintendo 3DS Edition

    1. Download the latest release of AKMenu-Next 3DS Edition.

    2. Extract the downloaded akmenu-next-3ds.zip file with 7-Zip.

      Warning

      If you are using TWiLight Menu++ on your Nintendo 3DS and want to also use AKMenu-Next, do not copy over BOOT.NDS as this will overwrite the one TWiLight Menu++ uses.

    3. From within the AKMenu-Next files, copy the following files/folders to your SD card root:

      • _nds folder
      • BOOT.NDS (Please read the warning above)
      • akmenu-next.cia

    4. Download the latest release of nds-bootstrap.

    5. Extract the downloaded nds-bootstrap.zip file with 7-Zip.

    6. Copy the contents of the files from the extracted nds-bootstrap.zip file to the _nds folder on your SD card.

    7. Boot up your Nintendo 3DS. Open the FBI app, go to your SD card, and find the akmenu-next.cia file. Select the cia, then choose "Install CIA".

    8. Launch the akmenu-next application from the 3DS home screen.

    Tip

    If you are upgrading from an older version of AKMenu-Next, or if AKMenu-Next does not appear on your 3DS home screen after installing the cia, restart your 3DS.

    Cheats

    1. If you'd like to be able to use cheats on your games, download a cheat database.

    2. You will need the usrcheat.dat file from the download link in the post. Copy this file to _nds/akmenunext/cheats/ on your SD card. (Create the cheats folder if it doesn't exist)

    Info

    AKMenu-Next, nds-bootstrap and Pico-Loader must be updated seperately of each other. If you wish to check for the latest version of AKMenu-Next, nds-bootstrap and Pico-Loader, you can view the latest versions & changelog from the Github releases linked here: AKMenu-Next Releases - nds-bootstrap Releases - Pico-Loader Releases

    Updating AKMenu-Next

    1. Go to the tab above for your applicable console/flashcart and Download the latest release of AKMenu-Next from there.

    2. Extract the downloaded akmenu-next-flashcart/dsi/3ds.zip file with 7-Zip.

    3. From within the extracted AKMenu-Next files, copy the _nds folder to your SD card root, clicking Replace files if on Windows or Linux, or Merge if on MacOS. Then follow the steps below.

    Please follow the below according to the system you are using:

    If you are on a flashcart

    • Copy & replace the BOOT.NDS file to your SD card root.

    If you are on a DSpico

    • Copy & replace the BOOT.NDS file to your SD card root.
    • Copy & Replace the _picoboot.nds file to your SD card root if you using a DSPico and are currently autobooting AKMenu-Next.

    If you are on a Nintendo DSi

    • Copy & replace the akmenu-next.dsi file to your SD card root.
    • If you do not also use Twilightmenu++, copy & replace the BOOT.NDS file to your SD card root. If you use Twilightmenu++, skip this file.
    • If you have AKMenu-Next installed as an app on HiyaCFW, copy over the title folder.

    If you are on a Nintendo 3DS

    • Copy & replace the akmenu-next.cia file to your SD card root.
    • If you do not also use Twilightmenu++, copy & replace the BOOT.NDS file to your SD card root. If you use Twilightmenu++, skip this file.
    • Install the akmenu-next.cia file with FBI, you may need to restart your 3DS console afterwards to use the new version.

    Updating nds-bootstrap

    1. Download the latest release of nds-bootstrap.

    2. Extract the downloaded nds-bootstrap.zip file with 7-Zip.

    3. Copy the contents of the files from the extracted nds-bootstrap.zip file to the _nds folder on your SD card.

    Updating Pico-Loader (Flashcarts Only)

    1. Download the latest release of Pico-Loader. Choose the version that corresponds with your flashcart.

    2. Extract the downloaded Pico_Loader_for_YOUR_FLASHCART_HERE.zip file with 7-Zip.

    3. Copy the contents of the files from the extracted Pico_Loader_for_YOUR_FLASHCART_HERE.zip file to the _pico folder on your SD card. Create this folder if it does not exist.
